# MyToDoApp

## Aplikacja MyToDoApp wraz z licznikiem czasu. Całość aplikacji oparta jest o bibliotekę React.

## Funkcjonalności, jakie ma aplikacja:
* dodawanie, usuwanie zadań,
* zakańczanie zadań,
* dodawanie, usuwanie operacji do zadań,
* rejestracja czasu operacji.

## Do ostylowania użyto Material UI
